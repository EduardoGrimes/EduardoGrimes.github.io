Uso de GitHub Pages

GitHub Pages es un servicio gratuito de alojamiento de sitios web que permite a los usuarios publicar páginas web directamente desde sus repositorios de GitHub. Es una forma sencilla y eficaz de crear sitios web para proyectos personales, portafolios, documentación y mucho más.
